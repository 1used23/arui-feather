# LazyRenderer

Компонент для отложенного рендера.

```javascript
import LazyRenderer from 'arui-feather/lazy-renderer';
```




## Props


| Prop  | Тип  | По умолчанию | Обязательный | Описание |
| ----- | ---- | ------------ | ------------ |----------|
| children | Node\|Array.<Node> |  |  |  |
| visible | Boolean |  |  | Показывать или нет |
